 SoundAI Landing Page
Project Overview
Welcome to the SoundAI Landing Page repository. This project is a modern, dark-themed hero page designed to showcase a music operations (music ops) management platform. It emphasizes a minimalist, high-contrast aesthetic with a focus on powerful typography and a unique visual texture/noise effect.

 Key Features
Dark Mode Native: A high-contrast, pure-black background design that's easy on the eyes.

Responsive Layout: Fully optimized for desktop, tablet, and mobile viewing using Tailwind CSS.

Unique Visuals: Features a striking, custom-styled hero image with a speckle/noise texture and a radial gradient effect.

Clear Call-to-Action (CTA): Prominent buttons (See how it works, Start listening) to guide user interaction.

Modern Navigation: Clean, accessible header with login and sign-up actions.
